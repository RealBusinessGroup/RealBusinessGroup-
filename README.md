# Real Business Group

Real Business Group is a modern marketplace app built for the Pi ecosystem. It allows users to buy and sell products, offer services, and communicate in real-time.

## Features
- Product listing and management
- Real-time chat
- Secure Pi payments
- User profiles and roles
- Services booking

## Technology
- Flutter Web
- Firebase (or backend)
- Pi Network integration
